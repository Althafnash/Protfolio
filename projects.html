BlackWave

I developed a real-time Chat App using Node.js, Express, and Socket.IO, designed to facilitate instant communication between multiple users. The application features a simple login system allowing users to enter a username before joining the chatroom. Messages are sent and received instantly in real-time, providing a seamless chat experience. Users can view a live list of connected participants, while automatic notifications alert when someone joins or leaves the chatroom. The chat feed also includes auto-scroll functionality, ensuring the latest messages are always visible. The app boasts a responsive design, making it accessible and user-friendly across both desktop and mobile devices. This project showcases my skills in real-time communication, backend development, and UI design, demonstrating my ability to build interactive and functional web applications from scratch.

Data Bank

I created Data Bank, a desktop-inspired knowledge management application that provides an organized and interactive interface for exploring information across multiple subjects. The app features a desktop-like UI, where users can navigate through “files” representing individual HTML pages containing knowledge on topics such as science, history, technology, and more. Each file opens seamlessly, allowing users to read content in a structured and visually engaging layout. The project demonstrates my skills in UI/UX design, front-end development, and information organization, as well as my ability to create a familiar and intuitive experience for users interacting with complex knowledge systems. Data Bank is designed to make learning and browsing information enjoyable, combining a desktop interface with modern web technologies to deliver a unique educational platform.
